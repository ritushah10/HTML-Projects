What is a link?
A link is a a code or instruction that connects one part of a program or an element in a list to another.

What is an active link?
An active link is when a user clicks on a link and when the website is open in another tab if the user goes and sees the link which the user clicked.

What is a visited link?
A visited link means that the website connected to the link has already opened using the link.

What is the difference between an active link and a visited link?
An active link is for link that has not been visited or clicked but a visited link is for a link that has been visited or clicked.
